##  О проекте

**Karaaj** — это первый язык программирования на карачаево-балкарском языке. Создан для того, чтобы каждый желающий мог изучать программирование на родном языке и писать код, используя привычные с детства слова.

Язык поддерживает все основные конструкции: переменные, условия, циклы, функции, ввод/вывод данных. Синтаксис максимально приближен к естественному карачаево-балкарскому языку.

---

##  Возможности

- ✅ **Переменные** — `айыр имя = значение`
- ✅ **Вывод на экран** — `чыгар выражение`
- ✅ **Ввод данных** — `сор "подсказка"`
- ✅ **Условный оператор** — `эгер ... анда ... болмаса ... бют`
- ✅ **Циклы** — `айлан ... ден ... ге чейин ... бют`
- ✅ **Функции** — `функция имя(параметры) ... кайтар ... бют`
- ✅ **Арифметика** — `+`, `-`, `*`, `/`
- ✅ **Сравнения** — `>`, `<`, `>=`, `<=`, `==`, `!=`
- ✅ **Логические значения** — `догъру`, `ялгъан`
- ✅ **Комментарии** — `# комментарий`

1. Скачайте последнюю версию установщика со страницы [Releases](https://github.com/thetemirbolatov-official/karaaj/releases)
2. Запустите `Karaaj_Setup.exe`

##  Как использовать

### REPL режим (интерактивная оболочка)

Просто запустите программу без параметров:

```bash
karaaj
```

Появится приглашение `karaaj >>>`, где можно вводить команды и сразу видеть результат:

```
karaaj >>> чыгар "Салам Дуния!"
Салам Дуния!
karaaj >>> айыр x = 10
karaaj >>> чыгар x * 2
20
```

Для выхода введите `выход` или нажмите `Ctrl+C`.

### Запуск файла

Создайте файл с расширением `.aj` и запустите:

```bash
karaaj программа.aj
```

### Двойной клик по файлу

После установки файлы с расширением `.aj` автоматически ассоциируются с Karaaj. Просто дважды кликните по файлу — он запустится и выполнится.

##  Примеры кода

### Пример 1: Привет, мир!

```aj
чыгар "Салам, Дуния!"
```

### Пример 2: Переменные и арифметика

```aj
айыр x = 10
айыр y = 5
айыр сумма = x + y
чыгар "x + y = "
чыгар сумма
```

### Пример 3: Условный оператор

```aj
айыр сан = 15

эгер сан > 10 анда
    чыгар "Сан 10-дан уллуду"
болмаса
    чыгар "Сан 10-дан гиччиди"
бют
```

### Пример 4: Цикл

```aj
айлан i = 1 ден 5 ге чейин
    чыгар i
бют
```

### Пример 5: Ввод данных

```aj
сор "Атынгы жаз: "
айыр ат = сор
чыгар "Салам, "
чыгар ат
```

### Пример 6: Функции

```aj
функция квадрат(сан)
    кайтар сан * сан
бют

айыр беш_квадрат = квадрат(5)
чыгар "5-ни квадраты: "
чыгар беш_квадрат
```

### Полный пример `hello.aj`

```aj
# Пример программы на Karaaj
айыр салам = "Дуниягъа салам!"
чыгар салам

# Арифметика
айыр x = 10
айыр y = 5
айыр сумма = x + y
чыгар "x + y = "
чыгар сумма

# Условия
эгер x > y анда 
    чыгар "x чонду"
болмаса 
    чыгар "y чонду"
бют

# Цикл
айлан i = 0 ден 5 ге чейин
    чыгар i
бют

# Функция
функция квадрат(сан)
    кайтар сан * сан
бют

айыр беш_квадрат = квадрат(5)
чыгар "5-ни квадраты: "
чыгар беш_квадрат
```
